💳 Online Fraud Detection Project

📌 Project Overview

This project uses machine learning to detect fraudulent online transactions. The model was trained on a dataset of financial transactions and predicts whether a given transaction is likely to be fraudulent.

🛠 Technologies Used

  • Python
  • Pandas
  • Scikit-learn
  • Logistic Regression
  • Joblib (for model saving)
